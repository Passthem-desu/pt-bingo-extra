# 小帕的 Bingo 扩展包

[Yet Another Bingo](https://horrific.dev/bingo/) 的扩展包

## 配置环境

安装一个含有 Yet Another Bingo 模组的 1.21.1 环境。

创建一个新的世界，创建前请在设置的数据包中，启用 bingo.zip 数据包，以创建一个含有大厅的世界。

打开存档文件夹，在 datapacks 文件夹 clone 本项目。

```bash
git clone https://github.com/Passthem-desu/pt-bingo-extra.git
```

重新进入存档或者键入 `/reload` 指令以加载数据包。

## 开发

在 Github 上 fork 此项目到你的 Github 仓库，在你的仓库创建完以后，创建 Pull Request 回到本项目。
